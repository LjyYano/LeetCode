## 题目描述

**🔗 题目**：[485. 最大连续1的个数](https://leetcode.cn/problems/max-consecutive-ones/)  
**🏷️ 标签**：`数组`  
**🟢 难度**：`简单`  

给定一个二进制数组 `nums`，计算其中最大连续 `1` 的个数。

**示例 1：**
```
输入：nums = [1,1,0,1,1,1]
输出：3
解释：开头的两个数字和最后的三个数字都是连续 1 ，所以最大连续 1 的个数是 3.
```

**示例 2：**
```
输入：nums = [1,0,1,1,0,1]
输出：2
```

**提示：**
- `1 <= nums.length <= 10^5`
- `nums[i]` 不是 `0` 就是 `1`

---

## 解题思路

### 一次遍历法

#### 📝 核心思想
遍历数组，统计连续1的个数，遇到0时重置计数器，并更新最大值。

#### 🛠️ 实现步骤
1. 初始化当前连续1的计数器 `count` 和最大值 `maxCount`
2. 遍历数组：
   - 如果是1，计数器加1
   - 如果是0，更新最大值并重置计数器
3. 遍历结束后再次更新最大值（处理数组以1结尾的情况）
4. 返回最大值

#### 🧩 示例分析
以示例 1 为例：`nums = [1,1,0,1,1,1]`

```
初始：count=0, maxCount=0

i=0: nums[0]=1, count=1, maxCount=1
i=1: nums[1]=1, count=2, maxCount=2
i=2: nums[2]=0, count=0, maxCount=2
i=3: nums[3]=1, count=1, maxCount=2
i=4: nums[4]=1, count=2, maxCount=2
i=5: nums[5]=1, count=3, maxCount=3

答案：3
```

### 优化写法

可以在遍历过程中直接更新最大值，避免最后的额外判断。

## 代码实现

```java
public class L0485_MaxConsecutiveOnes {
    // 方法一：标准写法
    public int findMaxConsecutiveOnes(int[] nums) {
        int maxCount = 0;
        int count = 0;
        
        for (int num : nums) {
            if (num == 1) {
                count++;
                maxCount = Math.max(maxCount, count);
            } else {
                count = 0;
            }
        }
        
        return maxCount;
    }
    
    // 方法二：另一种写法
    public int findMaxConsecutiveOnesV2(int[] nums) {
        int maxCount = 0;
        int count = 0;
        
        for (int num : nums) {
            if (num == 1) {
                count++;
            } else {
                maxCount = Math.max(maxCount, count);
                count = 0;
            }
        }
        
        // 处理数组以1结尾的情况
        return Math.max(maxCount, count);
    }
}
```

## 复杂度分析

- **时间复杂度**：O(n)，其中 n 是数组长度，只需要遍历一次数组。
- **空间复杂度**：O(1)，只使用了常数额外空间。
