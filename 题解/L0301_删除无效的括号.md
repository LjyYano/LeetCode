## 题目描述

**🔗 题目**：[删除无效的括号](https://leetcode.cn/problems/remove-invalid-parentheses/)  
**🏷️ 标签**：`广度优先搜索` `回溯` `字符串`  
**🔴 难度**：`困难`  

给你一个由若干括号和字母组成的字符串 s ，删除最小数量的无效括号，使得输入的字符串有效。返回所有可能的结果。答案可以按 任意顺序 返回。

**示例 1：**
```
输入：s = "()())()"
输出：["(())()","()()()"]
```

**示例 2：**
```
输入：s = "(a)())()"
输出：["(a())()","(a)()()"]
```

**提示：**
- 1 <= s.length <= 25
- s 由小写英文字母以及括号 '(' 和 ')' 组成
- s 中至多含 20 个括号

---

## 解题思路

### 回溯法

####📝 核心思想
首先计算需要删除的左括号和右括号数量，然后使用回溯法枚举所有可能的删除方案。
